PROJECTS
I have worked on multiple projects for coursework at RIT and a few side projects, which you can check out below. You can also check out most of these on my GitHub! Some of the projects listed here are not available on GitHub due to online presence restrictions for the respective code files.
  • Vehicular Route Guidance using Traffic Prediction. Implemented a route guidance system in Java by evaluating three different re-routing algorithms which use traffic prediction data to avoid traffic congestions. SUMO – Simulation of Urban Mobility was used for simulation of traffic, TraCI – Traffic Control Interface was used as a client for SUMO and TraaS – Traci as a Service, a java library for TraCI was used to generate and run the simulation. Click here to check out project files on GitHub.
  • Parallel Programming Solution to the Graph Coloring Problem. Implemented a parallel algorithm for the Graph Coloring problem in Java using the pj2 library, which colors the vertices of a graph in polynomial time but reduces the absolute running time proportional to the number of processors used to color the graph. Worked in a team of 2 for this project. Click here to check out project files on GitHub.
  • IoT Application for blind control. Worked in a team of 3 to design and developed an Android IoT application which reads light intensity and room temperature from sensors connected to a server running on Raspberry Pi and controls the blinds in the room as per the set fuzzy logic rules. An Android client was developed to give the user an UI which interacts with the server. The server was built using pi4j and JSONRPC2 java library was used for communication between Android application and the server.
  • Speck 32/64 block cipher & a reduced round attack. Implemented Speck 32/64 which is a block cipher with block size 32 bits and key size 64 bits, in Java. Click here to check out Speck 32/64 code on GitHub. Along with the implementation of Speck 32/64, a reduced 3-round attack was implemented on the same Speck 32/64 cipher. Click here to check out the code for the attack on Speck 32/64 on GitHub.
  • Improved Bully Algorithm. Implemented an improved version of Bully algorithm in Java, in a team of 3, which reduces the number of messages passed from O(n2) to O(n) where n is the number of nodes in the network, to find a process with the highest ID to be elected as the new leader for a shared resource in a distributed network. Java RMI was used for networking nodes. Click here to check out project files on GitHub.
  • Content Based Image Retrieval. Developed a Content Based Image Retrieval system, in a team of 2, using Wavelet Based Color Histogram method where Haar wavelets are used to compute horizontal, vertical and approximate coefficients, and then these coefficients are weighted and Histogram Intersection Distance method is used to identify similar images from a huge database of images. Matlab was used to develop the system. Click here to check out project files on GitHub.
  • Content Addressable Network (CAN). Built a structured Content Addressable Network, which is a p2p network, in Java, which uses Distributed Hash Table for searching files efficiently in the network. Java RMI was used for networking nodes. Click here to check out project files on GitHub.
  • TFTP Server & Client. Developed a TFTP Server and Client program in Java, which use UDP as the transport protocol. "connect", "get", "quit" and "?" commands were implemented in the Server and Client. Click here to check out project files on GitHub.
  • Gender-Neutral Restroom Finder. Developing an Android application to find gender-neutral restrooms on RIT campus. Google Maps API was used to get location of user and the restrooms, implemented in Java.
  • Climate Control System. Designed an expert system in Prolog which takes multiple temperature preferences from room occupants and sets the indoor temperature to please all occupants in an energy efficient manner. Click here to check out project files on GitHub.
